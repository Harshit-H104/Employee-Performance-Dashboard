# HR Analytics Dashboard

## Project Overview

This project presents an HR Analytics Dashboard developed using Power BI and PostgreSQL. The dashboard analyzes workforce data for 999 employees and provides insights into employee distribution, salary expenditure, hiring trends, department performance, work mode distribution, and employee attrition.

## Key Features

* Total Employees, Active Employees, and Resigned Employees KPIs
* Department-wise Employee Distribution
* Department-wise Salary Analysis
* Monthly Hiring Trend Analysis
* Work Mode Distribution (Remote vs On-site)
* Top 10 Highest Paid Employees
* Interactive Filters for Department, Location, and Work Mode

## Tools Used

* PostgreSQL
* Power BI
* SQL

## Business Insights

* Identified departments with the highest workforce and salary expenditure.
* Analyzed hiring trends across different months.
* Compared workforce distribution by work mode.
* Monitored employee attrition and workforce composition.
